Tech Stack

How are we creating the data you see on this website?

We use large language models to parse what we find on the internet.

Our tech stack includes:

  • Python Pydantic defines the data model.
  • Python FAST API serves the web pages.
  • Azure Startup Grant pays the cloud costs.
  • OpenSearch retrieves the studies.
  • OpenAI's language models to parse raw text.
  • HTMX to build the minimal front end.
  • Reddit to fetch personal stories.
  • Medarxiv to fetch clinical studies.
